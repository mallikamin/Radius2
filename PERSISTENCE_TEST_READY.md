## ❌ Failure Scenarios

### If rotation reverts to 285° in Step 5:
**Diagnosis steps:**
1. Check if you opened a different project (verify ID suffix)
2. Check if auto-generated map was opened instead of master
3. Check browser console for errors during project load
4. Check network tab for unexpected PUT requests during load

### If rotation shows different value but not 285° or your value:
**Possible causes:**
- Another user edited the map concurrently
- Sync-branches overwrite issue (check other annotations too)
- State initialization bug (compare DB vs UI on fresh load)
